def get_playbook(self, playbook_id: str) -> Optional[dict]:
    with self._conn() as conn:
        row = conn.execute("SELECT * FROM playbooks WHERE playbook_id = ?", [playbook_id]).fetchone()

Create backend/api/reports.py with router = APIRouter(prefix="/api/reports", tags=["reports"]).

**POST /api/reports/investigation/{investigation_id}**
- Accept body: InvestigationReportRequest (include_chat, include_playbook_runs)
- Fetch from SQLite via asyncio.to_thread: investigation_cases row for investigation_id, related detections, chat_messages if include_chat=True, playbook_runs for this investigation if include_playbook_runs=True
- Build content_json dict with keys: investigation (dict), detections (list), chat_history (list), playbook_runs (list)
- Title: f"Investigation Report: {investigation.get('title', investigation_id)}"
- Generate HTML string using Python f-string template (not Jinja2 — keep it simple). Template must include: report title as <h1>, generation timestamp, investigation summary table, timeline section with detections (rule_name, severity, created_at), chat history if present, playbook runs if present
- Call weasyprint.HTML(string=html_content).write_pdf() wrapped in asyncio.to_thread to get bytes
- Store report record via asyncio.to_thread(stores.sqlite.insert_report, {...}) with content_json=json.dumps(content_dict), store pdf_bytes in content_json under key "pdf_b64" as base64
- Return 201 JSONResponse with the report record (id, type, title, subject_id, created_at)

**POST /api/reports/executive**
- Accept body: ExecutiveReportRequest (period_start, period_end, title)
- Fetch from SQLite: COUNT detections in period, COUNT investigations in period (via asyncio.to_thread)
- Fetch from DuckDB: alert_volume, mttd/mttr from daily_kpi_snapshots if table exists (graceful fallback to zeros if table missing — it's created in plan 03)
- Build content_json with: period, alert_volume, detection_count, investigation_count, mttd_avg, mttr_avg
- Generate HTML for executive PDF: report title, period, KPI summary table, top detections by severity count
- Use asyncio.to_thread for WeasyPrint PDF generation
- Store and return 201 with report record

**GET /api/reports**
- Return list from asyncio.to_thread(stores.sqlite.list_reports)
- Strip pdf_b64 from content_json before returning (only return metadata)
- Return JSONResponse({"reports": [...]})

**GET /api/reports/{report_id}/pdf**
- Fetch report via get_report; 404 if not found
- Parse content_json, extract pdf_b64, base64 decode to bytes
- Return Response(content=pdf_bytes, media_type="application/pdf", headers={"Content-Disposition": f'attachment; filename="report-{report_id}.pdf"'})
