**Step 1 — Add two methods to SQLiteStore in backend/stores/sqlite_store.py.**

Both methods MUST use `with self._conn() as conn:` to avoid connection leaks (do NOT use a raw lambda over `_conn()` — that leaves connections open):

```python
def get_detection_techniques(self) -> list[dict]:
    """Return all detections with non-null attack_technique. Each dict has keys: attack_technique, attack_tactic."""
    with self._conn() as conn:
        rows = conn.execute(
            "SELECT attack_technique, attack_tactic FROM detections "
            "WHERE attack_technique IS NOT NULL AND attack_technique != ''"
        ).fetchall()
        return [dict(r) for r in rows]

def get_playbook_trigger_conditions(self) -> list[str]:
    """Return a flat list of all trigger_conditions JSON strings from all playbooks."""
    with self._conn() as conn:
        rows = conn.execute("SELECT trigger_conditions FROM playbooks").fetchall()
        return [r["trigger_conditions"] for r in rows]
```

**GET /api/analytics/mitre-coverage logic:**

1. Fetch detections via asyncio.to_thread:
   ```python
   rows = await asyncio.to_thread(stores.sqlite.get_detection_techniques)
   ```

2. Fetch playbook trigger conditions via asyncio.to_thread:
   ```python
   tc_jsons = await asyncio.to_thread(stores.sqlite.get_playbook_trigger_conditions)
   ```
   Parse each JSON array; any element matching regex `^T\d{4}` is treated as a technique reference.

3. Build coverage dict — for each (technique, tactic) pair from detections:
   - Normalise tactic: strip whitespace, lowercase
   - If tactic not in MITRE_TACTICS, bucket under "other"
   - Set sources=["detected"]

4. For each technique found in playbook trigger_conditions:
   - If technique already in coverage under any tactic, append "playbook_covered" to sources if not already present
   - If technique not yet seen, bucket under the tactic found in detections (or "other" if unknown), create entry with sources=["playbook_covered"]

5. Set status per technique: if "detected" in sources → "detected"; elif "hunted" in sources → "hunted"; elif "playbook_covered" in sources → "playbook_covered"; else → "not_covered"

6. Return JSONResponse:
   ```json
   {
     "tactics": [...MITRE_TACTICS...],
     "coverage": {
       "execution": {
         "T1059": {"sources": ["detected", "playbook_covered"], "status": "detected"}
       }
     }
   }
   ```
