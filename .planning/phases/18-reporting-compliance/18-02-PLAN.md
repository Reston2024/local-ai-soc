MITRE_TACTICS = [

**GET /api/analytics/mitre-coverage logic:**

1. Query detections table for all rows with non-null attack_technique:
   ```sql
   SELECT attack_technique, attack_tactic FROM detections
   WHERE attack_technique IS NOT NULL AND attack_technique != ''
   ```
   Wrap in asyncio.to_thread using lambda over sqlite_store._conn().execute().fetchall().

2. Query playbooks table for all trigger_conditions:
   ```sql
   SELECT trigger_conditions FROM playbooks
   ```
   Parse each JSON array; any element that looks like a MITRE technique ID (starts with "T" followed by digits, e.g. matches regex `^T\d{4}`) is treated as a technique reference.

3. Build coverage dict — for each (technique, tactic) pair from detections:
   - Normalise tactic: strip whitespace, lowercase
   - If tactic not in MITRE_TACTICS, bucket under "other"
   - Set sources=["detected"]

4. For each technique found in playbook trigger_conditions:
   - If technique already in coverage under any tactic, append "playbook_covered" to sources if not already present
   - If technique not yet seen, bucket under the tactic found in detections (or "other" if unknown), create entry with sources=["playbook_covered"]

5. Set status per technique: if "detected" in sources → "detected"; elif "hunted" in sources → "hunted"; elif "playbook_covered" in sources → "playbook_covered"; else → "not_covered"

6. Return JSONResponse:
   ```json
   {
     "tactics": [...MITRE_TACTICS...],
     "coverage": {
       "execution": {
         "T1059": {"sources": ["detected", "playbook_covered"], "status": "detected"}
       }
     }
   }
   ```
