<objective>
Pin the Caddy Docker image to an immutable digest to complete the P11-T02 deferred item.

Purpose: A mutable tag (`caddy:2.9-alpine`) can be silently updated by Docker Hub, introducing supply chain risk. An immutable digest guarantees the exact same image bytes are used every pull. This was deferred during Phase 11 because Docker Desktop was unavailable.

Output: docker-compose.yml updated with `caddy:2.9-alpine@sha256:<digest>`. Requires Docker Desktop running to obtain the digest — this plan has a human checkpoint.
</objective>
