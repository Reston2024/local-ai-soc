  <name>Task 1: Extend scripts/eval_models.py with embedding and reranker fields</name>
  <files>scripts/eval_models.py</files>
  <action>
Read scripts/eval_models.py in full to understand the current EvalResult dataclass and output structure.

Locate the EvalResult dataclass and add these fields (all Optional with default None — backward-compatible):
```python
embed_model: Optional[str] = None          # e.g. "bge-m3" or "mxbai-embed-large"
reranker_enabled: Optional[bool] = None    # True/False/None
embed_latency_ms: Optional[int] = None     # ms to produce embedding for the eval query
rerank_latency_ms: Optional[int] = None    # ms for reranker call (0 if disabled)
recall_at_5: Optional[bool] = None         # True if ground-truth passage in top-5 retrieved
```

In the eval run loop:
- Set embed_model from settings.OLLAMA_EMBED_MODEL
- Set reranker_enabled from settings.RERANKER_ENABLED
- Time the embed call using time.monotonic_ns() before/after; record (t_end - t_start) // 1_000_000 as embed_latency_ms
- If RERANKER_ENABLED: time the rerank_passages() call; record rerank_latency_ms. If disabled: record 0.
- For recall_at_5: after retrieval, check if any of the top-5 retrieved documents contains a known ground_truth substring. If no ground_truth field on the eval row: leave recall_at_5=None.

Update the markdown summary table printed to stdout to include columns: embed_model, avg_embed_latency_ms, reranker_enabled.

Verify backward compatibility: any existing eval_results.jsonl entries parse without errors (missing new fields just absent from JSON — no schema enforcement needed).
  </action>

  </verify>
  <done>--dry-run completes without errors. No Python tracebacks. Summary table is printed to stdout including embed_model and reranker_enabled columns.</done>
