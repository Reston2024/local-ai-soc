Create backend/services/reranker_service.py. This is a STANDALONE FastAPI app — NOT imported by the main backend app factory (main.py).

Implement:

**Imports and startup:**
- Import AutoTokenizer and AutoModelForSequenceClassification from transformers
- Import torch
- Detect device: device = "cuda" if torch.cuda.is_available() else "cpu"
- Load model at module startup (before app creation) using MODEL_NAME = "BAAI/bge-reranker-v2-m3"
- Call .to(device) and .train(False) on the model after loading

**Pydantic models:**
- RerankRequest(BaseModel): query: str, passages: list[str], top_k: int = 5
- RankedPassage(BaseModel): passage: str, score: float, original_index: int
- RerankResponse(BaseModel): ranked: list[RankedPassage]

**Endpoints:**
- POST /rerank: For each (query, passage) pair, tokenize with truncation=True max_length=512, run model, extract logits scalar as score. Sort by score descending. Return top_k results. Wrap blocking model inference in asyncio.to_thread().
- GET /health: Returns {"status": "ok", "model": MODEL_NAME, "device": device}

**App setup:**
Name the FastAPI instance "app": app = FastAPI(title="SOC Reranker", version="1.0.0")

Add if __name__ == "__main__" guard that calls uvicorn.run(app, host="0.0.0.0", port=8100).

Use backend.core.logging.get_logger(__name__) for structured logging (not print statements). Log model load, device, and VRAM usage at startup.

Follow error-handling patterns from backend/services/thehive_client.py.
