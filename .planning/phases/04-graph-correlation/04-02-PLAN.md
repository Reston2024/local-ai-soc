New models (exact field names and defaults per CONTEXT.md locked schema):
```python
class GraphNode(BaseModel):
    id: str
    type: str  # host|ip|domain|alert|process|user
    label: str
    attributes: dict = Field(default_factory=dict)
    first_seen: str = ""
    last_seen: str = ""
    evidence: list[str] = Field(default_factory=list)

class GraphEdge(BaseModel):
    id: str
    type: str  # dns_query|connection|alert_trigger|related_event|observed_on
    src: str
    dst: str
    timestamp: str = ""
    evidence_event_ids: list[str] = Field(default_factory=list)

class AttackPath(BaseModel):
    id: str
    node_ids: list[str]
    edge_ids: list[str]
    severity: str  # max alert severity; "info" if no alerts
    first_event: str
    last_event: str

class GraphResponse(BaseModel):
    nodes: list[GraphNode]
    edges: list[GraphEdge]
    attack_paths: list[AttackPath] = Field(default_factory=list)
    stats: dict = Field(default_factory=dict)
```

Helper _get_or_create_node(nodes, nid, type_, label, ev_id=None, timestamp=None) — creates node if absent, otherwise updates evidence list + first_seen/last_seen timestamps (min/max comparison as ISO strings).

_extract_nodes(events, alerts) -> dict[str, GraphNode]:
  - For each event dict: extract host (node host:{host}), src_ip (node ip:{src_ip}), dst_ip (node ip:{dst_ip}), query (node domain:{query} when event_type matches dns/dns_query)
  - Optional: if ev.get("user"): node user:{user}; if ev.get("process"): node process:{process}
  - For each alert dict: node alert:{alert.get("id")} with label=alert.get("rule","alert"), attributes={"severity": alert.get("severity","info"), "description": alert.get("description","")}
  - Use ev_id=ev.get("id") and timestamp=ev.get("timestamp") when calling _get_or_create_node

_extract_edges(events, alerts, nodes) -> list[GraphEdge]:
  - edge_count = 0 (increment for each edge, use as part of ID)
  - For each event: determine event_type via ev.get("event_type") or ev.get("event", "")
  - If src_ip and dst_ip: emit GraphEdge(id=f"e{edge_count}", type="connection", src=f"ip:{src_ip}", dst=f"ip:{dst_ip}", timestamp=ts, evidence_event_ids=[ev_id])
  - Elif host and dst_ip (no src_ip): emit GraphEdge(id=f"e{edge_count}", type="connection", src=f"host:{host}", dst=f"ip:{dst_ip}", timestamp=ts, evidence_event_ids=[ev_id])
  - If event_type in ("dns", "dns_query") and host and query: emit GraphEdge(id=f"e{edge_count}", type="dns_query", src=f"host:{host}", dst=f"domain:{query}", timestamp=ts, evidence_event_ids=[ev_id])
  - For each alert: find triggering event by alert.get("event_id"); determine target node (prefer host:{host} if available, else ip:{dst_ip}); emit GraphEdge(type="alert_trigger", src=f"alert:{alert.get('id')}", dst=target_nid, ...)

_group_attack_paths(nodes, edges) -> list[AttackPath]:
  - Union-Find pattern (copy from correlation/clustering.py but operate on node IDs not event IDs)
  - parent = {nid: nid for nid in nodes}
  - For each edge: union(edge.src, edge.dst) if both in parent
  - Group nodes by find(nid) root
  - For each component: collect edge_ids (edges where find(src)==root), compute severity via _path_severity, compute first_event/last_event from node.first_seen values
  - Return list[AttackPath] with id=f"path-{i+1}"

_path_severity(path_node_ids, nodes) -> str:
  - For each alert node in path (node.type == "alert"): get node.attributes.get("severity","info")
  - Return max by SEVERITY_RANK; default "info"

build_graph(events: list[dict], alerts: list[dict]) -> GraphResponse:
  - nodes = _extract_nodes(events, alerts)
  - edges = _extract_edges(events, alerts, nodes)
  - attack_paths = _group_attack_paths(nodes, edges)
  - Return GraphResponse(nodes=list(nodes.values()), edges=edges, attack_paths=attack_paths, stats={"node_count":len(nodes), "edge_count":len(edges), "event_count":len(events), "alert_count":len(alerts)})
