<task type="auto">
  <name>Task 1: Create .cmd wrappers for start, stop, status scripts</name>
  <files>scripts/start.cmd, scripts/stop.cmd, scripts/status.cmd</files>
  <action>
    Create three .cmd files. All three follow the same pattern: check if pwsh.exe
    is on PATH, and if so invoke the corresponding .ps1 with pwsh; otherwise print
    a clear error with an install command and exit 1.

    scripts/start.cmd:
    ```
    @echo off
    rem start.cmd — wrapper for start.ps1 (requires PowerShell 7 / pwsh)
    where pwsh.exe >nul 2>&1
    if %errorlevel% neq 0 (
        echo ERROR: PowerShell 7 ^(pwsh^) not found.
        echo Install it with:  winget install Microsoft.PowerShell
        echo Then re-run: scripts\start.cmd
        exit /b 1
    )
    pwsh -NoLogo -File "%~dp0start.ps1" %*
    ```

    scripts/stop.cmd:
    ```
    @echo off
    rem stop.cmd — wrapper for stop.ps1 (requires PowerShell 7 / pwsh)
    where pwsh.exe >nul 2>&1
    if %errorlevel% neq 0 (
        echo ERROR: PowerShell 7 ^(pwsh^) not found.
        echo Install it with:  winget install Microsoft.PowerShell
        echo Then re-run: scripts\stop.cmd
        exit /b 1
    )
    pwsh -NoLogo -File "%~dp0stop.ps1" %*
    ```

    scripts/status.cmd:
    ```
    @echo off
    rem status.cmd — wrapper for status.ps1 (requires PowerShell 7 / pwsh)
    where pwsh.exe >nul 2>&1
    if %errorlevel% neq 0 (
        echo ERROR: PowerShell 7 ^(pwsh^) not found.
        echo Install it with:  winget install Microsoft.PowerShell
        echo Then re-run: scripts\status.cmd
        exit /b 1
    )
    pwsh -NoLogo -File "%~dp0status.ps1" %*
    ```

    Use `%~dp0` (directory of the .cmd file) so the wrappers work when invoked
    from any working directory, not just the project root.
    The `%*` at the end forwards any additional arguments to the .ps1 script
    (e.g. -SkipBuild, -DevMode for start.ps1).
