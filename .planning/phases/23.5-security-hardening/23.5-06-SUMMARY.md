## Decisions Made

- Meta-detection rules use simple `detection.selection` + `condition: selection` pattern instead of `count()` aggregation with `timeframe`; pySigma 0.x does not support `timeframe` as a standard field, so aggregation semantics are described in comments and delegated to the collection layer.
- TOTP replay cache key is `SHA-256(operator_id:code)` to ensure uniform key length and avoid key injection.
- `prompt_hash` is computed from the FULL prompt (not the 64KB-truncated version) so integrity verification remains valid even when storage is capped.
