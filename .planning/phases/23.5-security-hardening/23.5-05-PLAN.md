**logging.py — Switch to TimedRotatingFileHandler (T10):**

In `setup_logging`, replace both `RotatingFileHandler` instances with `TimedRotatingFileHandler`.

For the main backend log handler:
```python
# Before:
file_handler = logging.handlers.RotatingFileHandler(
    log_path / "backend.jsonl",
    maxBytes=10 * 1024 * 1024,
    backupCount=5,
    encoding="utf-8",
)

# After:
file_handler = logging.handlers.TimedRotatingFileHandler(
    log_path / "backend.jsonl",
    when="midnight",
    interval=1,
    backupCount=30,  # 30 days retention
    encoding="utf-8",
    utc=True,
)
```

For the LLM audit log handler:
```python
# Before:
llm_audit_handler = logging.handlers.RotatingFileHandler(
    llm_audit_log,
    maxBytes=10 * 1024 * 1024,
    backupCount=5,
    encoding="utf-8",
)

# After:
llm_audit_handler = logging.handlers.TimedRotatingFileHandler(
    llm_audit_log,
    when="midnight",
    interval=1,
    backupCount=30,  # 30 days retention
    encoding="utf-8",
    utc=True,
)
```

`TimedRotatingFileHandler` is stdlib — no new packages required. The `utc=True` parameter
ensures rotation occurs at midnight UTC rather than local time, which is correct for a
server emitting UTC-timestamped logs.

Remove the `maxBytes` parameter — it does not exist on `TimedRotatingFileHandler`.
