**TEMPLATE_SHA256 (analyst_qa.py lines 65–72):**

```typescript
function confidenceLevel(score: number | undefined): string {
  if (score === undefined) return 'unknown'
  if (score >= 0.8) return 'high'
  if (score >= 0.5) return 'medium'
  return 'low'
}
```

Replace the chat message rendering block (lines 151–156). The new rendering adds:
