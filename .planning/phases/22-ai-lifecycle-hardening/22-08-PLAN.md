<objective>
Add GET /api/copilot/response/{audit_id} to the provenance router.

Purpose: The ROADMAP requires a dedicated endpoint at this path that returns the LLM response
record enriched with cited source details (grounding_event_ids, confidence_score, is_grounded).
The existing GET /api/provenance/llm/{audit_id} returns LlmProvenanceRecord (metadata only —
model_id, template SHA, response SHA). This new endpoint returns the caller-facing trust signals
needed by the UI and by API consumers auditing AI responses.

Design decision: The llm_audit_provenance table stores response_sha256 (hash) but NOT
response_text (the raw LLM output is not persisted for privacy/storage reasons). The new
endpoint therefore returns all available trust metadata: audit_id, model_id, grounding_event_ids,
confidence_score, is_grounded (derived: grounding_event_ids is non-empty), and created_at.
This satisfies the ROADMAP intent of "returns response + cited sources" via the trust signals.

Output: New endpoint at /api/copilot/response/{audit_id} and CopilotResponseRecord model.
</objective>

    ```python
    @router.get(
        "/copilot/response/{audit_id}",
        response_model=CopilotResponseRecord,
        summary="Get AI Copilot response trust record",
        description=(
            "Return the trust metadata for a given AI Copilot response: model ID, "
            "grounding event IDs (cited sources), confidence score, and whether the "
            "response was grounded in retrieved evidence. Note: response text is not "
            "stored — this endpoint surfaces trust signals for audit and UI display."
        ),
    )
    async def get_copilot_response(
        audit_id: str,
        request: Request,
        ctx: OperatorContext = Depends(require_role("analyst", "admin")),
        stores: Stores = Depends(get_stores),
    ) -> CopilotResponseRecord:
        row = await asyncio.to_thread(
            stores.sqlite.get_llm_provenance, audit_id
        )
        if row is None:
            raise HTTPException(status_code=404, detail="Copilot response not found")
        grounding_ids = row.get("grounding_event_ids") or []
        return CopilotResponseRecord(
            audit_id=row["audit_id"],
            model_id=row["model_id"],
            grounding_event_ids=grounding_ids,
            confidence_score=row.get("confidence_score"),
            is_grounded=len(grounding_ids) > 0,
            prompt_template_name=row.get("prompt_template_name"),
            operator_id=row.get("operator_id"),
            created_at=row["created_at"],
        )
    ```
