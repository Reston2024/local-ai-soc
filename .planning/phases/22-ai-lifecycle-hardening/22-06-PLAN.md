### Task 2: Human visual checkpoint

**Type:** `checkpoint:human-verify`

**What was built:**
- `GET /api/query/ask` now returns `audit_id`, `grounding_event_ids`, `is_grounded` in every response
- `/api/query/ask/stream` done-event includes `audit_id` and `is_grounded`
- `llm_audit_provenance` stores `confidence_score` (0.0–1.0 heuristic)
- `analyst_qa.SYSTEM` and `triage.SYSTEM` prefixed with `[AI Advisory — not a verified fact]`
- InvestigationView copilot panel: yellow AI Advisory banner + colour-coded confidence badge on every assistant message; badge and banner have no dismiss button
- SettingsView system tab: model-status card showing active model, last-known model, drift alert
- `GET /api/settings/model-status` endpoint accessible at analyst+ role
- Full eval harness at `tests/eval/` (6 prompt eval tests + grounding/confidence/drift/advisory unit tests)

**How to verify:**

1. Start the backend: `uv run uvicorn backend.main:app --reload --host 127.0.0.1 --port 8000`
2. Start the frontend: `npm run dev --prefix dashboard`
3. Navigate to an Investigation view. Send any question to the AI Copilot.
   - Confirm: yellow "AI Advisory" banner appears above the response text
   - Confirm: confidence badge appears (green/amber/red) — colour depends on whether context events were retrieved
   - Confirm: there is NO close/dismiss (×) button on either the banner or badge
   - Confirm: response text is italic and muted in colour
4. Navigate to Settings → System tab.
   - Confirm: "AI Model Status" card is shown (not the old placeholder text)
   - Confirm: active model name is displayed
5. Run `curl -s http://127.0.0.1:8000/api/query/ask -X POST -H "Content-Type: application/json" -H "Authorization: Bearer $AUTH_TOKEN" -d '{"question":"test"}' | python -m json.tool`
   - Confirm: response JSON includes `audit_id`, `grounding_event_ids`, `is_grounded`, `confidence_score` keys

**Resume signal:** Type "approved" to mark phase complete, or describe any issues found.

**Done:** All visual checks pass, no dismiss button present on advisory UI elements.
