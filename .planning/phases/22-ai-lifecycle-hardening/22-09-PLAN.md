<objective>
Wire model drift detection into the generate() and stream_generate() hot path in OllamaClient.

Purpose: The ROADMAP requires drift to be detected "on each LLM call" with a WARNING log.
Currently, drift only runs when an analyst manually visits Settings → System tab (GET /api/settings/model-status).
This gap closure adds a private _check_model_drift() method to OllamaClient that is called at the
start of generate() and stream_generate() before the HTTP call. The method is non-fatal — if
list_models() fails or the SQLite write fails, the LLM call proceeds normally.

The settings endpoint continues to work unchanged. The drift check there is retained as the
"authoritative" record — the hot-path check adds the WARNING log on mismatch but delegates the
full record_model_change + set_kv write to the settings endpoint when the analyst reviews it.
Rationale: Calling list_models() on EVERY generate() call adds one HTTP round-trip to Ollama.
To avoid performance impact, the hot-path check uses a cached last_known value read from SQLite
(one read per call) and only calls list_models() when no last_known exists yet. If last_known
exists and the configured model matches it, the check is a cheap SQLite read and returns quickly.

Output: _check_model_drift() in ollama_client.py, called from generate() and stream_generate().
</objective>

    ```python
    async def _check_model_drift(self, effective_model: str) -> None:
        """Check for model drift on each LLM call (P22-T04 hot-path check).

        Reads last_known_model from SQLite and compares to effective_model.
        Logs a WARNING if they differ. Writes the new model as last_known when
        no prior record exists. Always non-fatal — exceptions are swallowed.

        Args:
            effective_model: The model name being used for this call (after
                             model/use_cybersec_model resolution).
        """
        if self._sqlite is None:
            return
        try:
            import asyncio as _asyncio
            last_known: str | None = await _asyncio.to_thread(
                self._sqlite.get_kv, "last_known_model"
            )
            if last_known is None:
                # First call — seed last_known_model (non-fatal)
                await _asyncio.to_thread(
                    self._sqlite.set_kv, "last_known_model", effective_model
                )
                log.debug(
                    "Model drift: seeding last_known_model",
                    model=effective_model,
                )
            elif last_known != effective_model:
                log.warning(
                    "Model drift detected on LLM call",
                    last_known_model=last_known,
                    active_model=effective_model,
                )
        except Exception as exc:
            log.debug("Model drift check failed (non-fatal)", error=str(exc))
    ```
