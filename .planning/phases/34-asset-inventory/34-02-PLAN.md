SQLite assets DDL:
```sql
CREATE TABLE IF NOT EXISTS assets (
    ip         TEXT PRIMARY KEY,
    hostname   TEXT,               -- most recent, may be null
    tag        TEXT NOT NULL,      -- "internal" | "external"
    risk_score INTEGER DEFAULT 0,  -- recomputed at query time via JOIN
    last_seen  TEXT NOT NULL,      -- ISO8601
    first_seen TEXT NOT NULL       -- ISO8601, never updated
);
```

Upsert pattern (ON CONFLICT — update last_seen and hostname, preserve first_seen):
```sql
INSERT INTO assets (ip, hostname, tag, last_seen, first_seen)
VALUES (?, ?, ?, ?, ?)
ON CONFLICT(ip) DO UPDATE SET
    hostname  = COALESCE(excluded.hostname, assets.hostname),
    tag       = excluded.tag,
    last_seen = excluded.last_seen
```
