    **3. Create backend/services/osint.py:**

    Module docstring: "Passive OSINT enrichment service. Read-only lookups against public APIs. No active scanning."

    **Rate limiting — declare at module level (before the class):**

    ```python
    import asyncio

    # Per-source rate limiters — module-level singletons
    # VirusTotal free tier: 4 req/min → 1 per 15s to be safe
    _vt_lock = asyncio.Lock()
    _VT_INTERVAL = 15.0  # seconds between VirusTotal requests

    # AbuseIPDB free tier: 1000 checks/day → ~1 per 86s; use 90s for safety
    _abuse_lock = asyncio.Lock()
    _ABUSE_INTERVAL = 90.0

    # Shodan free tier: no hard rate limit stated but 1 req/s is safe
    _shodan_lock = asyncio.Lock()
    _SHODAN_INTERVAL = 1.0
    ```

    These locks enforce serialization; the sleep inside the lock enforces the inter-request interval.

    Implement helper functions:

    `_sanitize_ip(ip: str) -> str`:
    - Use `ipaddress.ip_address(ip)` to validate — raises ValueError("invalid IP") on failure
    - Reject private: `addr.is_private` → ValueError("private IP not enriched")
    - Reject loopback: `addr.is_loopback` → ValueError("loopback IP not enriched")
    - Return str(addr)

    `_is_cache_valid(fetched_at: str | None, ttl_hours: int = 24) -> bool`:
    - If fetched_at is None: return False
    - Parse ISO timestamp, compare to now(UTC), return True if age < ttl_hours

    `@dataclass OsintResult`:
    - Fields: ip: str, whois: dict | None, abuseipdb: dict | None, geo: dict | None, virustotal: dict | None, shodan: dict | None, cached: bool = False, fetched_at: str = ""

    Implement `class OsintService`:
    ```python
    def __init__(self, sqlite_store, settings_obj=None):
        self._store = sqlite_store
        self._settings = settings_obj or settings  # from backend.core.config import settings
    ```

    `async def enrich(self, ip: str) -> OsintResult`:
    1. Call `_sanitize_ip(ip)` — let ValueError propagate (API layer returns 400)
    2. Check cache: `cached = await asyncio.to_thread(self._store.get_osint_cache, ip)`
       If cached and `_is_cache_valid(cached.get("fetched_at"))`: return OsintResult(**cached["data"], cached=True, fetched_at=cached["fetched_at"])
    3. Run all lookups concurrently via `asyncio.gather(*tasks, return_exceptions=True)`:
       - `_whois(ip)` — always runs (no key required)
       - `_abuseipdb(ip)` — skip if ABUSEIPDB_API_KEY is ""
       - `_geo(ip)` — use local mmdb file
       - `_virustotal(ip)` — skip if VT_API_KEY is ""
       - `_shodan(ip)` — skip if SHODAN_API_KEY is ""
    4. Wrap each result: if exception → log warning and set field to None
    5. Build OsintResult, compute fetched_at = now(UTC).isoformat(), expires_at = (now + 24h).isoformat()
    6. Persist to cache
    7. Return result

    Implement lookup methods (all return dict or None on exception):

    `async def _whois(self, ip: str) -> dict | None`:
    - `import whois` (python-whois); call `await asyncio.to_thread(whois.whois, ip)`
    - Extract: registrar, creation_date (as ISO string), expiration_date, country, org
    - Return dict with those fields (None values OK)
    - On any exception: return None

    `async def _abuseipdb(self, ip: str) -> dict | None`:
    - Acquire `_abuse_lock`, then sleep `_ABUSE_INTERVAL` seconds inside the lock before making the request.
      Pattern:
      ```python
      async with _abuse_lock:
          await asyncio.sleep(_ABUSE_INTERVAL)
          # ... make the httpx request ...
      ```
    - GET https://api.abuseipdb.com/api/v2/check?ipAddress={ip}&maxAgeInDays=90
    - Headers: {"Key": self._settings.ABUSEIPDB_API_KEY, "Accept": "application/json"}
    - Use httpx.AsyncClient(timeout=10)
    - Extract: abuseConfidenceScore, totalReports, countryCode, usageType, isp, domain
    - On non-200 or exception: return None

    `async def _geo(self, ip: str) -> dict | None`:
    - `import geoip2.database`
    - mmdb_path = self._settings.GEOIP_DB_PATH
    - If file does not exist: log warning once, return None
    - `reader = geoip2.database.Reader(mmdb_path)`
    - `response = await asyncio.to_thread(reader.city, ip)`
    - Extract: country_name, country_iso_code, city, latitude (response.location.latitude), longitude (response.location.longitude), autonomous_system_number, autonomous_system_organization
    - Close reader. On geoip2.errors.AddressNotFoundError: return {"country_name": None, "latitude": None, "longitude": None}
    - On any other exception: return None

    `async def _virustotal(self, ip: str) -> dict | None`:
    - Acquire `_vt_lock`, then sleep `_VT_INTERVAL` seconds inside the lock before making the request.
      Pattern:
      ```python
      async with _vt_lock:
          await asyncio.sleep(_VT_INTERVAL)
          # ... make the httpx request ...
      ```
    - GET https://www.virustotal.com/api/v3/ip_addresses/{ip}
    - Headers: {"x-apikey": self._settings.VT_API_KEY}
    - Use httpx.AsyncClient(timeout=15)
    - Extract from response.json()["data"]["attributes"]["last_analysis_stats"]: malicious, suspicious, harmless, undetected
    - Also extract: country, as_owner, reputation
    - On non-200 or exception: return None

    `async def _shodan(self, ip: str) -> dict | None`:
    - Acquire `_shodan_lock`, then sleep `_SHODAN_INTERVAL` seconds inside the lock before making the request.

    - On shodan_lib.exception.APIError (including "no information available"): return None
    - On any exception: return None
