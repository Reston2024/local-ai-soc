```python
class NormalizedEvent(BaseModel):
    event_id: str                   # uuid4
    timestamp: Union[datetime, str] # ISO-8601 from EVE record
    ingested_at: Union[datetime, str]
    source_type: Optional[str]      # "suricata_eve"
    source_file: Optional[str]      # file_path arg
    hostname: Optional[str]         # record.get("host") — sensor hostname
    src_ip: Optional[str]           # record["src_ip"]
    src_port: Optional[int]         # record.get("src_port")
    dst_ip: Optional[str]           # record["dest_ip"]  ← EVE uses dest_ip not dst_ip
    dst_port: Optional[int]         # record.get("dest_port")
    domain: Optional[str]           # dns.rrname or http.hostname
    url: Optional[str]              # http.url
    event_type: Optional[str]       # mapped from EVE event_type (see mapping below)
    severity: Optional[str]         # mapped from alert.severity (1=critical..4=low)
    detection_source: Optional[str] # alert.signature
    attack_technique: Optional[str] # alert.metadata.mitre_attack_id[0]
    attack_tactic: Optional[str]    # alert.metadata.mitre_tactic_name[0]
    raw_event: Optional[str]        # json.dumps(record)[:8192]
    tags: Optional[str]             # comma-separated: "sid:N,category:X,flow_id:N"
    network_protocol: Optional[str] # record["proto"]
    event_outcome: Optional[str]    # alert.action ("allowed"→"success","blocked"→"failure")
    case_id: Optional[str]
```

<!-- EVE event_type → NormalizedEvent event_type mapping -->
```python
_EVE_TYPE_MAP = {
    "alert": "detection",
    "flow": "network_connect",
    "dns": "dns_query",
    "http": "network_connect",
    "tls": "network_connect",
    "heartbeat": "heartbeat",
}
# Unknown event types → use EVE type string as-is

_EVE_SEVERITY_MAP = {1: "critical", 2: "high", 3: "medium", 4: "low"}
# Default severity for non-alert events: "info"

_EVE_ACTION_MAP = {"allowed": "success", "blocked": "failure"}
```

```python
metadata = alert.get("metadata") or {}

tactic = (metadata.get("mitre_tactic_name") or [None])[0]
```

Key implementation details:

1. **Module-level constants** (see interfaces):
   - `_EVE_TYPE_MAP`, `_EVE_SEVERITY_MAP`, `_EVE_ACTION_MAP`
   - `_MAX_RAW = 8 * 1024`

2. **`_parse_eve_timestamp(ts_str: str) -> datetime`** helper:
   - Use `datetime.fromisoformat(ts_str.replace("Z", "+00:00"))` for ISO-8601 with timezone offset
   - Fall back to `datetime.now(timezone.utc)` if parsing fails

3. **`SuricataEveParser.parse_record(record: dict, source_file: str = "suricata_eve", case_id: str | None = None) -> NormalizedEvent | None`**:
   - Extract common fields: timestamp, src_ip, dest_ip→dst_ip, src_port, dest_port→dst_port, proto→network_protocol, host→hostname
   - Determine event_type using `_EVE_TYPE_MAP.get(record.get("event_type", ""), record.get("event_type"))`
   - For `alert` events:
     - Extract `alert = record.get("alert", {})`
     - `severity = _EVE_SEVERITY_MAP.get(alert.get("severity", 4), "low")`
     - `detection_source = alert.get("signature")`
     - `event_outcome = _EVE_ACTION_MAP.get(alert.get("action", ""), None)`
     - MITRE extraction using the pattern in interfaces
     - Tags: `f"sid:{alert.get('signature_id')}"`, `f"category:{alert.get('category')}"` (filter None/empty)
   - For `dns` events:
     - `domain = record.get("dns", {}).get("rrname")`
     - Add `f"dns_type:{record.get('dns', {}).get('rrtype', '')}"` to tags
   - For `http` events:
     - `domain = record.get("http", {}).get("hostname")`
     - `url = record.get("http", {}).get("url")`
     - Add `f"method:{record.get('http', {}).get('http_method', '')}"` to tags
   - For `flow` events:
     - `flow = record.get("flow", {})`
     - `event_outcome = "success"` if flow.get("state") == "closed" else None
   - Add `f"flow_id:{record.get('flow_id')}"` to tags if flow_id present
   - `tags = ",".join(filter(None, tag_parts))` or None
   - `raw_event = json.dumps(record, default=str)[:_MAX_RAW]`
   - Return NormalizedEvent with `event_id=str(uuid4())`; `source_type="suricata_eve"`

4. **`SuricataEveParser.parse(file_path, case_id=None) -> Iterator[NormalizedEvent]`**:
   - Open file with `encoding="utf-8", errors="replace"`
   - For each non-empty stripped line: `json.loads(line)` in try/except json.JSONDecodeError (log WARNING, continue)
   - Call `parse_record(record, source_file=file_path, case_id=case_id)` and yield if not None

5. **`supported_extensions: list[str] = []`** — programmatic use, not extension-based.
