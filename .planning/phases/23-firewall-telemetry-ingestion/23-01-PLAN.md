<!-- Regex patterns from RESEARCH.md -->
```python
import re
_RFC3164_RE = re.compile(
    r'^(?P<month>\w{3})\s+(?P<day>\d{1,2})\s+(?P<time>\d{2}:\d{2}:\d{2})\s+'
    r'(?P<host>\S+)\s+kernel:\s+(?P<body>.+)$'
)
_KV_RE = re.compile(r'(\w+)=(\S*)')
_PREFIX_RE = re.compile(r'^(\S+)\s+IN=')

# Zone mapping (interface name -> zone tag)
_ZONE_MAP = {"green0": "green", "red0": "red", "blue0": "blue", "orange0": "orange"}

# Action mapping (prefix -> event_outcome)
_ACTION_MAP_SUCCESS = {"FORWARDFW", "INPUTFW"}  # starts with these -> success
# Anything else (DROP_*, REJECT_*) -> failure

# Severity mapping (all firewall events are info by default)
_SEVERITY_DEFAULT = "info"
_DROP_SEVERITY = "medium"  # dropped packets warrant medium severity
```

Create `ingestion/parsers/ipfire_syslog_parser.py` implementing IPFireSyslogParser(BaseParser).

Key implementation details:

1. **Module-level constants:**
   - `_RFC3164_RE` regex for syslog header (see interfaces above)
   - `_KV_RE` for key=value pairs in the iptables body
   - `_PREFIX_RE` to extract the log prefix (FORWARDFW, DROP_CTINVALID, etc.)
   - `_ZONE_MAP = {"green0": "green", "red0": "red", "blue0": "blue", "orange0": "orange"}`
   - `_MAX_RAW = 8 * 1024` (same as osquery_parser.py)

2. **`_parse_timestamp(month, day, time_str) -> datetime`** helper:
   - Format: `datetime.strptime(f"{month} {day:>2} {time_str}", "%b %d %H:%M:%S")`
   - Add current year via `.replace(year=datetime.now().year)`
   - If parsed date is more than 30 days in the future: subtract 1 year (handles Dec→Jan)
   - Return timezone-aware datetime with `timezone.utc`

3. **`_extract_action(prefix: str) -> str`** helper:
   - Returns `"success"` if prefix in `{"FORWARDFW", "INPUTFW"}`
   - Returns `"failure"` otherwise (DROP_*, REJECT_*, etc.)

4. **`_extract_severity(prefix: str) -> str`** helper:
   - Returns `"info"` for FORWARDFW/INPUTFW
   - Returns `"medium"` for DROP_* prefixes
   - Returns `"medium"` for REJECT_* prefixes

5. **`IPFireSyslogParser.parse_line(raw_line, source_file="ipfire_syslog", case_id=None) -> NormalizedEvent | None`**:
   - Strip the line; return None if empty
   - Match `_RFC3164_RE`; return None if no match
   - Extract `host`, `body` from match groups
   - Match `_PREFIX_RE` against body to get the log prefix; return None if no match
   - Extract all key=value pairs from body using `_KV_RE`
   - Build tags list: `[f"in:{kv.get('IN','')}", f"out:{kv.get('OUT','')}"]` (filter empty values)
   - Add zone tags: for each interface name in IN/OUT that's in _ZONE_MAP, add `f"zone:{_ZONE_MAP[iface]}"`
   - `src_port` = `int(kv['SPT'])` if 'SPT' in kv else None
   - `dst_port` = `int(kv['DPT'])` if 'DPT' in kv else None
   - Return NormalizedEvent with all fields populated; `event_id=str(uuid4())`

6. **`IPFireSyslogParser.parse(file_path, case_id=None) -> Iterator[NormalizedEvent]`**:
   - Open file with `encoding="utf-8", errors="replace"`
   - For each non-empty stripped line, call `parse_line(line, source_file=file_path, case_id=case_id)`
   - Yield the result if not None
   - Log warnings on parse errors (non-fatal)

7. **`supported_extensions: list[str] = []`** — not extension-based, used programmatically.

Set `source_type = "ipfire_syslog"`.
