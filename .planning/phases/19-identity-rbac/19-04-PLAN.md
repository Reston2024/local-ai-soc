Create backend/api/operators.py with an APIRouter:

Endpoints (all require valid auth via verify_token at router level; write
endpoints additionally require require_role("admin")):

POST /api/operators
  - Body: OperatorCreate
  - Admin only: Depends(require_role("admin"))
  - Generate: operator_id = str(uuid4()), raw_key = generate_api_key()
  - Store: await asyncio.to_thread(stores.sqlite.create_operator, operator_id, body.username, hash_api_key(raw_key), key_prefix(raw_key), body.role)
  - Return: OperatorCreateResponse with api_key=raw_key (the ONLY time it is returned)
  - Status: 201

GET /api/operators
  - Admin + analyst allowed: Depends(require_role("admin", "analyst"))
  - Return: {"operators": [OperatorRead(...)]} list — NO hashed_key, NO totp_secret

DELETE /api/operators/{operator_id}
  - Admin only: Depends(require_role("admin"))
  - Call: await asyncio.to_thread(stores.sqlite.deactivate_operator, operator_id)
  - Prevent self-delete: if operator_id == request.state.operator.operator_id → 400

POST /api/operators/{operator_id}/rotate-key
  - Admin only: Depends(require_role("admin"))
  - Generate new raw_key = generate_api_key()
  - await asyncio.to_thread(stores.sqlite.update_operator_key, operator_id, hash_api_key(raw_key), key_prefix(raw_key))
  - Return: OperatorRotateResponse with new api_key (the ONLY time it is returned)

POST /api/operators/{operator_id}/totp/enable
  - Admin only (enabling your own TOTP or another admin's)
  - Generate: secret = generate_totp_secret()
  - Store secret: UPDATE operators SET totp_secret=? WHERE operator_id=?
    (add set_totp_secret(operator_id, secret) method to SQLiteStore)
  - uri = get_provisioning_uri(secret, username)
  - Return: {"qr_code": totp_qr_png_b64(uri), "provisioning_uri": uri}
  - Status: 200

DELETE /api/operators/{operator_id}/totp
  - Admin only
  - Clear: UPDATE operators SET totp_secret=NULL WHERE operator_id=?
  - Return: {"detail": "TOTP disabled"}
